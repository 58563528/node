# weibo

> 微博插件

```js
module.exports = {
  plugin: {
    './plugin/weibo': {},
  },
}
```

用户发送 "微博/热搜/微博热搜/WB/RS/WBRS", 机器人抓取微博热搜 TOP10 返回

![image](https://user-images.githubusercontent.com/8413791/115852493-b8d65c00-a45a-11eb-9127-3c3e5f55ec27.png)
